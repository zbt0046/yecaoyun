## 三条线路到底怎么选？

这是很多人纠结的地方，我帮你捋一下。

野草云香港机房目前有三种网络线路：优质BGP、精品BGP、国际BGP。它们的核心区别在于回程路由走法不同。

优质BGP的回程路由大致是：电信走CMI、联通走CU/NTT/CMI均衡、移动走Level3。实测大陆方向延迟大概在40-80ms之间，电信用户体验最好。这条线路带宽给得大，价格也最实惠，是大多数用户的首选。

精品BGP的回程是：电信走联通CU、联通走CU、移动走CMI。三网回程都走优质链路，整体更稳定，但带宽比优质BGP小。适合对网络质量要求高、能接受小带宽的业务场景。

国际BGP不含CMI和CU，线路择优选路，大陆方向延迟会比较高（100ms+），但国际带宽大且便宜，适合做跨境电商独立站、面向东南亚市场的业务。

简单来说：做大陆业务选优质BGP或精品BGP，做海外业务选国际BGP。
